# 시나리오 2: 금리 인하 시 은행/부동산 섹터 분석

## 개요
오후 2시, "한은 기준금리 0.25% 인하" 속보 발생 시 즉각 대응하는 실전 워크플로우

**소요 시간**: 초기 대응 3분 + 심층 분석 30분 + 종가 후 정리 1시간
**목적**: 금리 변동이 은행/부동산 섹터에 미치는 영향 신속 파악 및 투자 액션

---

## Phase 1: 즉시 대응 (첫 3분)

### 14:00:00 - 속보 확인
```
📱 알림: "한국은행, 기준금리 3.50% → 3.25% (0.25%p 인하)"
```

### 14:00:30 - 첫 번째 체크 (90초)

**1. 국채 금리 즉시 확인**
```
체크 항목:
- 3년물 국고채 수익률
- 10년물 국고채 수익률
- 수익률 곡선 (Yield Curve)
```

**해석 로직:**
```
한은 인하 -0.25%
    ↓
국채 금리가...

Case A: -8bp 이상 하락
→ 시장이 "긍정적" 해석 (추가 인하 기대)
→ 은행/부동산 호재

Case B: 변동 없거나 상승
→ 시장이 "이게 마지막 인하" 해석
→ 또는 인플레 우려 (부정적)
→ 은행/부동산 제한적 반응 예상
```

**⚠️ 초보자 함정:**
> "금리 인하 = 무조건 은행 악재" (X)
> 실제로는 **국채 시장 반응**이 먼저!

**2. 섹터 지수 즉시 확인 (60초)**
```
- KRX 은행지수: 실시간 변동률
- 건설/부동산 지수: 실시간 변동률
- 개별 종목: 대형 은행 3사, 대형 건설 3사
```

**첫 반응 해석:**
```
은행 +2% 이상: 시장이 긍정적 (NIM 우려 < 부실률 개선)
은행 -1% 이상: 시장이 부정적 (NIM 악화 우려)
건설 +3% 이상: PF 리스크 완화 기대감
```

## Phase 2: 심층 분석 (30분)

### 14:05 - 은행 섹터 임팩트 분해 (15분)

**A. NIM (순이자마진) 영향 계산**

```
분석 체크리스트:

1. 현재 은행권 평균 NIM: ____%
2. 예금 금리 vs 대출 금리 민감도:
   - 예금 금리가 더 빨리 하락? OR
   - 대출 금리가 더 빨리 하락?

3. 예상 NIM 변화:
   기준금리 -0.25%
   → 예금금리 -0.20% (80% 전가)
   → 대출금리 -0.25% (100% 전가)
   → NIM: -5bp 예상
```

**실무 경험칙:**
```
현재 NIM이...

1.8%p 이상 (건강): -5bp는 감내 가능
1.5%p 미만 (취약): -5bp도 부담 큼

→ 개별 은행별 NIM 수준 체크 필수
```

**B. 대출 성장 가능성**

```
체크 포인트:

1. 가계대출 규제 현황
   - DSR 규제 수준: _____%
   - 규제 완화 시그널 있나?

   IF 규제 강함:
   → 금리 내려도 대출 못 늘림
   → 은행 수혜 제한적

2. 기업대출 수요
   - 최근 기업 투자심리: _____
   - 경기 회복 국면? OR 방어 국면?

   IF 경기 침체 국면:
   → 기업도 대출 안 받음
   → 대출 성장 제한적
```

**C. 자산건전성 (부실률)**

```
✅ 긍정적 시나리오:
금리 인하
→ 기업/가계 이자 부담 완화
→ 부실률 개선 (특히 PF, 자영업자)
→ 대손충당금 적립 부담 ↓
→ 은행 이익 ↑

⚠️ 주의 시나리오:
경기 침체 국면 인하라면:
→ 부실 악화 우려가 더 클 수도
→ 금리 효과 제한적
```

**종합 판단 (2024년 말 기준 예시):**

```
🤔 상황 판단:
"지금 인하는 경기 방어적 성격 강함"
→ 인플레가 잡혀서가 아니라, 경기 안 좋아서

📊 결론:
- NIM 하락 우려 (-5bp)
- 대출 성장 제한적 (규제 여전)
- 부실률 개선 기대 (+소폭)

→ 은행주 반응: 중립~약보합 예상
```

---

### 14:20 - 부동산 섹터 분석 (15분)

**A. 건설/시행사 (PF 중심)**

```
1. PF 만기 구조 체크:
   - 향후 6개월 만기: _____억
   - 향후 1년 만기: _____억

2. 금리 인하 효과:
   금리 -0.25%
   → PF 이자 부담: -0.2% 정도

   예: 1조원 PF → 연 20억 절감
   (큰 효과는 아님)

3. 핵심은 "롤오버(연장) 가능성"
   금리 하락
   → 은행 입장에서 연장 부담 ↓
   → PF 만기 협상 유리

   → 단기 유동성 위기 완화!
```

**B. 분양 시장 영향**

```
1. 주택담보대출 금리 예상 변화:
   기준금리 -0.25%
   → 국채 금리 -8bp
   → 주담대 금리 -10~15bp (1~2주 후 반영)

2. 수요 반응 예측:
   월 상환액 변화:
   - 3억 대출, 30년, 4.5% → 4.35%
   - 월 152만원 → 149만원 (-3만원)

   체감 효과: 크지 않음
   BUT 심리적 "바닥 신호"로 작용 가능

3. 지역별 차이:
   - 서울/수도권: 관망 수요 움직임 가능 (40%)
   - 지방: 효과 제한적 (재고 과잉)
```

**C. 거래량 선행지표 모니터링**

```
즉시 체크할 지표:
1. KB부동산 Liiv - 주간 실거래 데이터
2. 네이버/다음 부동산 검색량 트렌드
3. 청약홈 방문자 수
4. 부동산 앱 다운로드 수

→ 2주 후부터 변화 관찰
```

**종합 판단 예시:**

```
📊 건설/부동산 전망:

✅ 단기 긍정:
- PF 롤오버 부담 완화 (유동성 위기 해소)
- 심리적 바닥 신호

⚠️ 중장기 신중:
- 실제 거래량 증가는 2-3개월 후 확인 필요
- 지역별 격차 심함
- 규제(DSR) 여전히 강력

→ 투자: PF 부담 적은 대형사 선별 매수
```

## 핵심 인사이트

### 1. 국채 시장이 선생님
```
금리 인하 발표
    ↓
국채 금리 반응 먼저 체크!
    ↓
국채가 하락 → 시장이 긍정적 해석
국채가 상승 → 시장이 부정적 해석
```

### 2. "이론 vs 실제" 항상 체크
```
이론: 금리 인하 = 은행 악재 (NIM 하락)
실제: 상황에 따라 다름
  - 부실률 개선 기대 > NIM 우려
  - 대출 성장 기대
  - 경기 사이클 어디쯤?
```

### 3. 타이밍 랙 이해
```
금리 인하 발표 (D-day)
    ↓
+1주: 은행 대출금리 변경 발표
    ↓
+2주: 실제 주담대 금리 반영
    ↓
+1개월: 가계대출 취급액 ��화
    ↓
+2개월: 부동산 거래량 변화

→ 너무 빨리 결론 내지 말 것!
```

### 4. 맥락이 모든 것을 결정
```
같은 금리 인하라도...

인플레 잡혀서 내린 거면:
→ 경기 연착륙 기대 (긍정)

경기 안 좋아서 내린 거면:
→ 방어적 인하 (중립~부정)

어느 쪽인지 판단이 핵심!
```
