# 시나리오 1: 반도체 섹터 주간 점검

## 개요
월요일 아침, 반도체 섹터 투자 담당자가 주간 정례 점검을 수행하는 실제 워크플로우

**소요 시간**: 약 1시간
**목적**: 주간 투자 전략 수립 및 모니터링 포인트 파악

---

## 분석 워크플로우

### 8:30 - 뉴스 스캔 (5분)

**체크 항목:**
- 주말 동안 미국/유럽/아시아 반도체 관련 뉴스
- 지정학적 이슈 (미중 갈등, 수출 규제 등)
- 주요 기업 공시 (실적 프리뷰, 증설 계획)
- **필라델피아 반도체 지수(SOX)** 금요일 장 마감 후 움직임

**중요 포인트:**
> 반도체는 글로벌 섹터라 미국 시장 반응이 한국 시장에 선행 지표

---

### 8:35 - 수급 데이터 체크 (10분) ⭐핵심

**체크 순서:**

1. **WSTS(세계반도체통계기구) 월간 출하 데이터**
   - 전월 대비 증감률
   - 전년 동기 대비 증감률

2. **북미 반도체 장비 BB Ratio (Book-to-Bill)**
   - 1.0 기준: 1.0 넘으면 수요 > 공급
   - 추세 확인 (상승/하락)

3. **TSMC, 삼성전자 가동률 추정치**
   - 증권사 리포트 참고
   - 업계 소스 정보

4. **DRAM/NAND 현물 가격 추이**
   - DRAMeXchange 등 가격 트래킹 사이트
   - 주간 변동률

**분석 로직:**
```
IF BB Ratio 하락 AND 출하량 증가
→ 공급 과잉 초기 신호 (위험)

예: 2018년 하반기
- "데이터센터 수요 폭발" 낙관론 팽배
- BUT BB Ratio 이미 0.9대로 하락
- 결과: 6개월 후 급락
```

---

### 8:45 - 재고 사이클 판단 (15분)

**핵심 지표:**

1. **재고일수 (DOI - Days of Inventory)**
   - 증가 추세? 감소 추세?
   - 업계 평균과 비교

2. **주요 고객사 재고 수준**
   - 애플, 삼성, Dell 등 실적 발표 시 재고 턴 체크
   - 엑셀에 시계열 데이터 정리

3. **유통 채널 재고**
   - PC/스마트폰 유통 재고 수준

**실무 인사이트:**
> **애플 재고일수가 2주 연속 늘어나면?**
> → 3개월 뒤 반도체 주문 감소 신호

**타이밍 랙:**
- 고객사 재고 증가 → 3-6개월 후 반도체 발주 감소
- 재고 소진 → 2-4개월 후 발주 증가

---

### 9:00 - 밸류에이션 체크 (10분)

**확인 지표:**
- **현재 PER vs 5년 평균 PER**
- **PBR** (반도체는 자산집약적 산업)
- **Forward P/E 대비 EPS 성장률** (PEG Ratio)

**⚠️ 주의사항:**

반도체는 **사이클 산업**이므로 밸류에이션 해석이 역설적:

```
일반 상식: PER 낮으면 싸다 → 매수
반도체 실전: PER 낮으면 사이클 정점 → 위험

일반 상식: PER 높으면 비싸다 → 매도
반도체 실전: PER 높으면 사이클 바닥 → 기회
```

**실제 사례:**
- 2019년 초: PER 10배 "싸다" → 실제로 바닥 (매수 기회)
- 2021년: PER 30배 "비싸다" → 한참 더 상승

**결론:**
> 밸류에이션은 마지막에 참고만 하고, **사이클 위치가 더 중요**

---

### 9:10 - 시나리오 구성 (20분)

**데이터 종합 예시 (2024년 12월 가정):**

```
✅ 수집된 데이터:
- BB Ratio: 1.15 (수요 > 공급)
- DRAM 가격: 6주 연속 상승 (+15% YoY)
- TSMC 가동률: 85% → 90%로 상승
- 재고일수: 업계 평균 10주 → 8주로 감소
- PER: 25배 (5년 평균 20배)

🤔 판단:
업사이클 중반부 진입 중
- 가격 ↑, 가동률 ↑, 재고 ↓ (모두 긍정)
- BUT PER 25배는 이미 많이 반영됨

📊 시나리오 및 확률:
```

| 시나리오 | 확률 | 조건 | 예상 수익률 (3개월) |
|----------|------|------|---------------------|
| **Base Case** | 60% | AI 서버 수요 지속 | +10% |
| **Bull Case** | 25% | AI 수요 서프라이즈 | +20% |
| **Bear Case** | 15% | 중국 경기 둔화 | -10% |

**투자 의견:**
- 현재 포지션 유지
- 추가 매수는 조정 시 (-5% 이상)
- 익절 타이밍: +15% 도달 시 일부 차익 실현

---

### 9:30 - 액션 아이템 정리 (5분)

**주간 모니터링 포인트:**

```
✓ 수요일 - 미국 ISM 제조업 지수 발표
  → 반도체 수요 선행지표
  → 50 이상 유지하는지 체크

✓ 목요일 - 삼성전자 잠정 실적 발표
  → 시장 예상치 대비 서프라이즈 여부
  → 컨퍼런스콜에서 메모리 업황 가이던스

✓ 금요일 - 엔비디아 주가 동향
  → AI 반도체 섹터 센티먼트 지표
  → SOX 지수 상관관계 체크
```

**위험 시나리오:**
```
⚠️ 모니터링 필요:
1. 미중 갈등 재점화
   - 특히 장비/소재 수출 규제 강화
   - 대만 지정학 리스크

2. 중국 부동산 추가 악화
   → PC/스마트폰 수요 타격
   → 소비자 가전 전반 영향

3. 금리 재인상 리스크
   → 성장주 밸류에이션 부담
   → PER 디레이팅
```

## 핵심 인사이트

### 1. 데이터 보는 순서가 중요
```
공급(BB Ratio) → 수급 밸런스(가격) → 재고 사이클
→ 밸류에이션 → 시나리오 구성
```

### 2. 선행 지표에 집중
- BB Ratio: 6개월 선행
- 고객사 재고: 3개월 선행
- 밸류에이션: 동행 또는 후행 (참고만)

### 3. 사이클 위치 파악이 핵심
```
바닥 → 회복 → 확장 → 정점 → 하락

현재 위치?
- 재고 추세
- 가격 추세
- 가동률 추세
→ 3가지가 모두 같은 방향이면 확신
```

### 4. 역발상 사고
```
"너무 싸 보이면" → 정점 근처일 수 있음
"너무 비싸 보이면" → 바닥 근처일 수 있음

사이클 산업은 밸류에이션 역설 주의!
```
