# 실전 분석 시나리오

이 폴더는 산업/섹터 분석가가 실무에서 사용하는 구체적인 워크플로우와 판단 로직을 시나리오 형태로 정리한 문서들입니다.

## 📚 시나리오 목록

### 1. [반도체 섹터 주간 점검](01_semiconductor_weekly_check.md)
**상황**: 월요일 아침 정례 섹터 점검
**소요시간**: 1시간
**핵심 학습**:
- BB Ratio, DRAM 가격 등 수급 데이터가 가장 중요
- 재고 사이클 판단 방법
- 사이클 산업의 밸류에이션 역설

### 2. [금리 인하 시 은행/부동산 분석](02_interest_rate_cut_banking_realestate.md)
**상황**: 한은 기준금리 0.25% 인하 속보 발생
**소요시간**: 즉시 3분 + 심층 30분 + 종가 후 1시간
**핵심 학습**:
- 국채 금리 반응이 선행 지표
- NIM vs 부실률 개선 트레이드오프
- PF 롤오버 이슈 분석

### 3. [에너지 섹터 - 유가 급등 시 대응](scenario_03_energy_oil_price_surge.md)
**상황**: 중동 지정학 리스크로 WTI +14.6% 급등
**소요시간**: 긴급 대응 15분 + 심층 분석 1시간
**핵심 학습**:
- 에너지 서브섹터 차별화 (통합석유 vs E&P vs 정유)
- 교차 섹터 영향 (항공↓, 화학↓)
- 크랙스프레드, 리그 카운트 등 실전 지표

### 4. [헬스케어 섹터 - FDA 신약 승인 뉴스](scenario_04_healthcare_fda_approval.md)
**상황**: 알츠하이머 신약 정규 승인, 주가 +18% 급등
**소요시간**: 장중 대응 30분 + 주말 딥다이브 3시간
**핵심 학습**:
- 임상 데이터 상세 분석 (Primary Endpoint, 안전성)
- Peak Sales 모델링
- 바이오텍 섹터 파급효과

### 5. [필수소비재 섹터 - 인플레이션 환경](scenario_05_consumer_staples_inflation.md)
**상황**: CPI +4.2% 인플레이션 재가속
**소요시간**: 당일 분석 2시간 + 주간 추적
**핵심 학습**:
- 원가 전가 능력에 따른 차별화
- Gross Margin 추이가 핵심
- Trading Down 징후 포착

### 6. [산업재 섹터 - 제조업 PMI 발표 후 대응](scenario_06_industrials_pmi_response.md)
**상황**: ISM PMI 46.2 급락, 경기 침체 문턱
**소요시간**: 긴급 대응 10분 + 심층 분석 1.5시간
**핵심 학습**:
- PMI 46대의 역사적 의미
- 서브섹터별 민감도 (기계 타격, 방산 방어)
- 턴어라운드 시그널 체크

### 7. [섹터 로테이션 - 경기 사이클 전환기](scenario_07_sector_rotation_cycle_shift.md)
**상황**: Late Cycle → Recession 전환 징후 포착
**소요시간**: 분기별 전략 재점검 4시간
**핵심 학습**:
- 4단계 경기 사이클 프레임워크
- LEI 활용한 현재 위치 진단
- 섹터별 사이클 최적화 전략

## 🎯 시나리오 활용 방법

### 학습 목적
1. **순차적 학습**: 1번부터 7번까지 순서대로 읽기
2. **상황별 참조**: 실제 비슷한 이벤트 발생 시 해당 시나리오 찾아보기
3. **체크리스트 활용**: 각 시나리오의 템플릿을 복사해서 실전 사용

### 실전 적용
- 각 시나리오의 **체크리스트**를 인쇄/복사해서 준비
- 실제 이벤트 발생 시 시나리오 워크플로우 따라가기
- 자신만의 노트 추가하며 개선

### 에이전트 시스템 개발
- 이 시나리오들이 LLM 에이전트의 **사고 과정 참고 자료**
- 에이전트가 어떤 순서로 데이터를 수집하고 판단해야 하는지 가이드
- Tool 개발 시 어떤 데이터가 필요한지 파악

## 💡 각 시나리오의 공통 구조

모든 시나리오는 다음 구조를 따릅니다:

```
1. 개요
   - 상황 설정
   - 소요 시간
   - 목적

2. 단계별 워크플로우
   - Phase 1: 긴급 파악 (5-15분)
   - Phase 2: 심층 분석 (30-60분)
   - Phase 3: 전략 수립
   - Phase 4: 모니터링 계획

3. 구체적 체크리스트
   - 데이터 소스
   - 확인 순서
   - 판단 로직

4. 핵심 인사이트
   - 경험칙
   - 주의사항
   - 함정

5. 실전 템플릿
   - 복사 가능한 분석 시트
```

## 🔧 에이전트 개발 시 활용

자율 에이전트 시스템 구축 시 이 시나리오들을 다음과 같이 활용:

### 1. Tool 개발 가이드
각 시나리오에서 언급된 데이터 소스 → Tool로 구현

예: 시나리오 #1에서 사용하는 데이터
- `fetch_bb_ratio()` - 반도체 BB Ratio
- `fetch_dram_price()` - DRAM 현물가
- `fetch_tsmc_capacity()` - TSMC 가동률

### 2. 프롬프트 엔지니어링
시나리오의 "판단 로직" → LLM 시스템 프롬프트에 반영

예:
```python
system_prompt = """
반도체 분석 시:
1. BB Ratio를 먼저 확인 (1.0 기준)
2. 가격 추세 확인 (DRAM/NAND)
3. 재고 사이클 판단
4. 밸류에이션은 마지막 (사이클 산업 특성)
"""
```

### 3. 워크플로우 자동화
시나리오의 단계별 체크리스트 → Agent의 실행 순서

### 4. 검증 로직
시나리오의 "주의사항" → 에이전트의 판단 검증
