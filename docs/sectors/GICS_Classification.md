## 교차 섹터 & 테마별 분류

### 클라우드 컴퓨팅
- 섹터: IT + 통신서비스
- 주요 기업: AWS(IT), Google Cloud(통신서비스), Microsoft Azure(IT)

### 사이버보안
- 섹터: IT
- 네트워크 보안, 엔드포인트 보안, 클라우드 보안

### 인공지능/머신러닝
- 교차 섹터: IT + 헬스케어 + 산업재
- AI 칩, AI 소프트웨어, AI 응용 서비스

### 핀테크
- 교차 섹터: IT + 금융
- 결제, 대출, 투자, 보험테크

### 이커머스
- 교차 섹터: IT + 경기소비재
- 온라인 마켓플레이스, 소셜커머스

### ESG/클린에너지
- 교차 섹터: 에너지 + 유틸리티 + 소재
- 탄소포집, 수소, 친환경 소재, 재생에너지

### 메타버스
- 교차 섹터: 통신서비스 + IT
- AR/VR 하드웨어, 메타버스 플랫폼, 공간컴퓨팅

### 블록체인/암호화폐
- 섹터: IT (+ 금융)
- 블록체인 인프라, 암호화폐, NFT, DeFi

### 로보틱스/자동화
- 교차 섹터: 산업재 + IT
- 산업용 로봇, 서비스 로봇, 자동화 소프트웨어

### 양자컴퓨팅
- 섹터: IT
- 양자 컴퓨터, 양자 소프트웨어, 양자 암호화

### 우주산업
- 섹터: 산업재 (항공우주/방위)
- 위성 발사, 우주 탐사, 위성 인터넷
